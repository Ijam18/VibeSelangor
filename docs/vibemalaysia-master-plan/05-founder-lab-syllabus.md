# 05 - Stage C Syllabus: Founder Lab (12 Weeks, Invite-Only)

## Founder Lab Objective
Move validated MVP teams toward investable startup readiness.

## 12-Week Structure
## Block 1 (Weeks 1-4): Traction System
- Define growth thesis
- Build repeatable acquisition loop
- KPI dashboard setup

## Block 2 (Weeks 5-8): Business Model & Unit Economics
- Pricing experiments
- CAC/LTV assumptions
- Revenue model stress tests

## Block 3 (Weeks 9-12): Fundability & Scale Readiness
- Pitch narrative refinement
- Data room preparation
- Investor simulation panel

## Required Outputs
1. Traction dashboard with weekly updates
2. Customer evidence pack
3. Financial assumptions sheet
4. Pitch deck (problem, solution, traction, model, ask)
5. Founder readiness review report

## Exit Gates
- Evidence of traction growth trend
- Team execution reliability
- Investor panel readiness score
