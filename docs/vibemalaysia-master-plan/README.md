## Locked Decisions
- Scope: Malaysia pilot-first
- Tracks: Juniors (9-12), Youth (13-24), Adults/SMEs (25+)
- Language: BM+EN bilingual delivery
- Evaluation philosophy: No-Codes-Only Vibes (Functionality + Design + Potential)

## How to Use
- Start with `01` and `07` to set operations and staffing.
- Run Stage A using `03` and track assessments from `06`.
- Progress qualified learners to Stage B via `04`, and Stage C via `05`.
- Use `08` for audits, governance, and reporting cadence.
- Use CSV templates for Sheets/DB ingestion.
